# 公募基金行业数据看板

中国证券投资基金业协会公募基金行业数据可视化看板，含两大模块：

- **市场总览**：公募基金资产净值/数量/管理机构数走势（2020-02 ~ 至今，月度）
- **销售机构榜**：公募基金销售保有规模榜单（2021Q1 ~ 至今，季度+半年度）

## 在线访问

https://kelvinye2020-arch.github.io/amac-fund-dashboard/

## 数据来源

- [中国证券投资基金业协会](https://www.amac.org.cn/)
- [市场月报](https://www.amac.org.cn/sjtj/tjbg/gmjj/)
- [销售机构榜](https://www.amac.org.cn/sjtj/datastatistics/fundsalesindustrydata/)

## 看板特性

### 市场总览
- 资产净值/管理机构/基金数量 跨6年走势
- 2025-11 口径调整标记（封闭/开放式 → 6 大类）
- 时间窗切换：滚动12月 / 近6月 / 全部
- 报告期切换驱动 KPI 卡片重算

### 销售机构榜
- 全市场 100 家机构完整明细
- 4 大业务洞察 KPI：TOP10 集中度 / 第三方独立销售占比 / 股票型指数被动化率 / 全市场权益规模
- TOP20 机构横向条形（3 字段切换：权益基金 / 非货币 / 股票型指数）
- 头部机构 4 期排名变迁凹凸图
- 2024H1 口径调整提示（股票+混合 → 权益基金）
- 报告期与市场总览智能映射（月份→半年/季度）

## 数据频率

- 市场总览：月度披露，协会通常在次月下旬发布
- 销售机构榜：
  - 2021Q1 ~ 2023Q4：季度披露
  - 2024H1 起：半年披露
